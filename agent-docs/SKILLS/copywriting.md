# Copywriting Skill — TED Friend

## 1. Purpose
This skill defines copywriting rules for the TED Friend project.

The system includes two audiences:
- students in grades 4–6;
- instructors using the dashboard.

Copy must match the audience, learning context, and bilingual nature of the project.

## 2. General Rule
Do not change existing copy unless the user explicitly asks for it or the requested UI change requires it.

When copy must be added or updated, keep it short, clear, and consistent with the existing product tone.

## 3. Student-Facing English
Student-facing English must be:
- simple;
- friendly;
- encouraging;
- age-appropriate for grades 4–6;
- short enough to avoid cognitive overload;
- supportive rather than judgmental.

Avoid:
- complex grammar;
- long paragraphs;
- abstract explanations;
- harsh correction language;
- overly formal academic wording.

## 4. Student-Facing Hebrew
Hebrew copy that addresses the student must be gender-inclusive.

Prefer neutral plural or inclusive forms when natural.

Examples of acceptable direction:
- `בחרו נושא להרצאה`
- `אפשר להמשיך לשלב הבא`
- `נסו שוב מאוחר יותר`

Avoid masculine-only or feminine-only phrasing when addressing the user.

## 5. Instructor-Facing Copy
Instructor-facing copy should be:
- professional;
- concise;
- practical;
- clear enough for quick scanning.

The instructor dashboard should not use overly childish wording.

## 6. Error Messages
Error messages should be calm and user-friendly.

Rules:
- Do not expose raw technical errors.
- Do not mention internal API failures to children.
- Explain what the user can do next.
- For AI failure, use the approved message:
  `Ted Friend לא יכול לענות כרגע, נסו שוב מאוחר יותר.`

## 7. Button Labels
Button labels should be:
- short;
- action-oriented;
- clear;
- consistent across the interface.

Avoid vague labels when the action can be clearer.

## 8. Hebrew-English Balance
Rules:
- Do not mix Hebrew and English unnecessarily.
- Keep English practice content in English unless the system already uses Hebrew support copy.
- Keep interface guidance consistent with the current screen language.
- Do not translate product names such as TED Friend unless explicitly requested.

## 9. Personal Data Warnings
Where relevant, login or student entry flows should clearly warn students not to share unnecessary personal information.

The warning should be simple and non-frightening.

## 10. Main Rule
Copywriting should help students and instructors understand what to do next without adding stress, clutter, or confusion.
